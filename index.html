Gomatrix.

Gomatrix is an open source linear albebra library for the Go programming language. Gomatrix has a focus on simple and efficient syntax to perform the functions it offers. It is written entirely in Go and is licenced under the GPL v3.0 liscence.

Types

Gomatrix adds some new types to Go, namely Matrix and Vector. Matrix can be used to define a mathematical matrix in Go. Each matrix has a 3 fields, Rows, Cols (uint) and Data ([]float64). Rows and Cols describe the dimensions of the matrix and data the contents.

type Matrix struct {
Rows uint,
Cols uint,
Data []float64,
}

Vector is similar to Matrix except it has no need for dimensions as the Vector can only be 1 dimesnional. For this reason the Vector struct is:

type Vector struct {
Data []float64,
}

Operations

Gomatrix can perform many operations such as the basic arithmatic for vectors and matrices.

Multiplication

Gomatrix can perform many kinds of multiplication. For example, dot finds the multiple of two matrices

func (matrix1 Matrix) Dot(matrix2 Matrix) (result Matrix, err error) {}

Dot takes two matrices in the form A.Dot(B) and returns an answer and an error, if it has worked the value of err should be nil. Gomatrix can also do element wise multiplication with MulElm (MulElm has the same inputs and returns as Dot). In addition Gomatrix can transform a vector by a matrix with MatDotVec.

func (matrix Matrix) MatDotVec(vector Vector) (result Vector, err error)

This function can be used to transform a cartesean point by some matrix by using the point as vector with 2 entries.
